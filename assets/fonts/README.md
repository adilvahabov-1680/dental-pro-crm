# PDF fonts

`DejaVuSans.ttf` / `DejaVuSans-Bold.ttf` — извлечены из npm-пакета
[`dejavu-fonts-ttf`](https://www.npmjs.com/package/dejavu-fonts-ttf) v2.37.3
(Bitstream Vera-производный шрифт, поддерживает кириллицу/латиницу с
диакритикой, включая азербайджанские ə/ş/ğ/ı/İ). Лицензия — см. `LICENSE`
в этой папке (Bitstream Vera License + Arev Fonts License): свободное
воспроизведение/распространение разрешено при сохранении уведомления об
авторстве; запрещена только продажа шрифта как самостоятельного продукта
и переименование модифицированных версий в "Bitstream"/"Vera".

Файлы закоммичены в репозиторий (`lib/pdf.ts` читает их по пути
`assets/fonts/...` относительно корня проекта), а не подключены через
`node_modules/dejavu-fonts-ttf` — Vercel-деплой не гарантированно бандлит
бинарные ассеты, на которые код ссылается динамической строкой пути
(`fs`-чтение внутри pdfkit, не статический `require`/`import`), поэтому
автоматический file-tracing их пропускал (см. `docs/DOCUMENTS.md`).
`next.config.ts` (`outputFileTracingIncludes`) дополнительно объявляет
эту папку явно для всех маршрутов — на случай, если сам факт нахождения
файла в дереве проекта тоже не попадёт в трассировку.
